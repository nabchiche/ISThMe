What is ISThMe?
ISThMe is the acronym of Illustrating Strategies for Therapeutic and Medical prescription

It offers an integrated software architecture allowing prescribers to simulate medical strategies on a virtual population. Through ISThMe, different strategies can be simulated and then compared.


User Manuel

How to launch the program?
SWindows: 
	* Run run.bat
Linux or Mac OS	
	* Run ISThMe/ISThMe.jar

How to run the simulations? Type the following:
Step 1: Choose the parameters:
Either use the default settings
Or use the custom settings.
Step 2: Choose if you want to visualize the results in files.
Step 3: Choose if you want to consider the risk when deciding the medical status of an individual
Step 4: In case in step 1 you have selected custom parameters, you have to enter the parameters of your choice.

Step 5: Choose the individual file to use if you do not want to use the default individual file of the program.

Execution:
Step 6: To run the strategies, click on the Recommended Strategy and Alternative Strategies buttons, depending on your choice

Step 7: To display the Histograms and curves, you must have executed both Strategies (otherwise the obtained results will be incorrect).
